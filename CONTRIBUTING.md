# Contribuții

Contribuțiile sunt binevenite.

1. creați un fork;
2. creați o ramură descriptivă;
3. nu adăugați chei API, exporturi reale sau date personale;
4. adăugați teste pentru regulile modificate;
5. rulați `pytest -q`;
6. executați notebookul cu date de test;
7. descrieți clar efectul asupra formulelor, deduplicării și compatibilității;
8. deschideți un pull request.

Modificările regulilor de eligibilitate sau punctaj trebuie să indice sursa
oficială, data consultării și comisia vizată. Contribuitorul confirmă că are
dreptul de a oferi contribuția sub licențele proiectului.
